## Overage Pricing Examples

### Pro Plan ($25/month)
- **Included**: 100,000 requests
- **Additional requests**: $0.0003 each
- **Example**: 150,000 requests = $25 + (50,000 × $0.0003) = **$40/month**

### Ultra Plan ($75/month)
- **Included**: 1,000,000 requests
- **Additional requests**: $0.0001 each
- **Example**: 1,500,000 requests = $75 + (500,000 × $0.0001) = **$125/month**

<Aside type="tip">
  All overage charges are calculated automatically by RapidAPI and billed monthly. No surprise bills!
</Aside>
